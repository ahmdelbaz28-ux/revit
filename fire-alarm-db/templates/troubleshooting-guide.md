# 🔥 Fire Alarm System - Troubleshooting Guide

---

## 1. COMMON PROBLEMS & SOLUTIONS

### 🔴 Fire Alarm Panel Issues

| Problem | Possible Cause | Solution |
|---------|---------------|----------|
| Panel shows " Trouble" | Wiring issue | Check all connections |
| Panel shows " Ground Fault" | Ground wire issue | Isolate circuits one by one |
| Panel won't power on | Power supply | Check 120V input, fuses |
| Batteries not charging | Battery connection | Check battery terminals |
| Display blank | LCD failure | Contact manufacturer |

### 🔴 Detection Issues

| Problem | Possible Cause | Solution |
|---------|---------------|----------|
| Detector in trouble | Dirty detector | Clean or replace |
| Detector won't reset | Wrong type | Verify compatibility |
| False alarms | Location issue | Check for steam, dust |
| Detector not responding | Address issue | Re-address device |

### 🔴 Notification Issues

| Problem | Possible Cause | Solution |
|---------|---------------|----------|
| Strobes not flashing | Wiring | Check polarity |
| Horns not sounding | Voltage low | Check wire size |
| Partial activation | Circuit overload | Reduce devices |
| Constant alarm | Ground fault | Find ground fault |

## 2. TESTING PROCEDURES

### Monthly Tests:
1. Check panel normal indication
2. Verify AC power on
3. Check battery indicator
4. Test central station signal (by phone)

### Annual Tests:
1. **Smoke Detector Test**
   - Use calibrated smoke test chamber
   - Verify alarm within 60 seconds
   - Record sensitivity reading

2. **Manual Station Test**
   - Pull handle
   - Verify alarm on panel
   - Reset properly

3. **Notification Test**
   - Activate alarm
   - Measure sound level (85 dBA min)
   - Verify strobe flash

4. **Battery Test**
   - Disconnect AC power
   - Place system in alarm
   - Verify 15 minutes operation

## 3. WIRING FAULT FINDING

### Troubleshooting Flow:

```
[Trouble Indication]
        │
        ▼
Check Panel Documents
        │
        ▼
Identify Circuit/Zone
        │
        ▼
Isolate at Panel
        │
        ├── Good → Problem in field
        │
        └── Bad → Problem at panel
```

### Common Wire Faults:

| Fault | Test | Fix |
|-------|------|-----|
| Open circuit | Ohm each wire | Find break, splice |
| Short circuit | Megger test | Find short, repair |
| Ground fault | Ground test | Isolate ground |
| Polarity reversed | Volt meter | Rewire |

---

## 4. ADDRESSABLE SYSTEM ISSUES

### Device Not Responding:

1. Check device address
2. Verify wire polarity
3. Check for loop breaks
4. Verify panel compatibility

### Multiple Device Failures:

1. Check loop wiring
2. Check for water damage
3. Verify power supply
4. Check for ground faults

---

## 5. NETWORK SYSTEM ISSUES

### Communication Failure:

1. Check network wiring
2. Verify termination resistors
3. Check fiber connections
4. Verify IP settings

---

## 6. EMERGENCY PROCEDURES

### System Failure:

1. Notify building management
2. Establish fire watch
3. Contact fire department
4. Document all incidents

### False Alarms:

1. Investigate cause immediately
2. Document in log
3. Notify central station
4. Take corrective action
